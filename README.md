Sham Expenses - PWA
--------------------
How to use:
1. Upload the contents of this folder to GitHub Pages root or any static host.
2. Open the hosted URL in Safari (iPhone) then use "Add to Home Screen" to install the PWA.
Features included:
- Purple and Black/Gold themes with toggle
- Monthly data per YYYY-MM stored in localStorage
- Add/edit/delete expenses, dynamic categories (add unlimited)
- Extra income added to total money
- Charts (daily & category) using Chart.js
- Print support
- Warning if expense exceeds average + threshold (20,000 IQD)
- Currency: Iraqi Dinar (د.ع)
- Service worker caches assets for offline use after first load
